## 目标

完成所有不可变/可变实体的 Rust 类型定义 + SQLite 持久化层，确保数据层可独立测试。

## 任务清单

### 1.1 项目初始化与 Cargo.toml [1h]

- [ ] 创建 Cargo 项目，配置 `Cargo.toml` 核心依赖：
  ```toml
  [dependencies]
  rusqlite = { version = "0.32", features = ["bundled"] }
  blake3 = "1.5"
  serde = { version = "1", features = ["derive"] }
  serde_json = "1"
  rkyv = { version = "0.8", features = ["validation"] }
  thiserror = "2"
  chrono = "0.4"
  uuid = { version = "1", features = ["v7"] }
  ```
- [ ] 初始化 `src/lib.rs`、`src/main.rs`、`src/error.rs`、`src/core/mod.rs`

### 1.2 公共类型（`core/types.rs`）[0.5h]

- [ ] `ContentId([u8; 32])` — Blake3 内容哈希包装类型，实现 `Copy + Clone + Eq + Hash`
- [ ] 类型别名：`SnapshotId = ContentId`, `DeltaId = ContentId`, `CheckpointId = ContentId`, `BackupId = ContentId`
- [ ] 内容寻址函数：`fn from_content(data: &[u8]) -> Self`

### 1.3 不可变实体定义 [2h]

按 [核心数据模型 §2.2](../architecture/02-核心数据模型.md) 定义：

- [ ] `FileNode` — `file_path: PathBuf`, `base_hash: Blake3Hash`，实现 `Hash + Eq`
- [ ] `SourceType` — `Manual | Agent(AgentInstanceId) | Backup`
- [ ] `LineDiff` — `hunks: Vec<Hunk>`
- [ ] `Hunk` — `old_start/old_len/new_start/new_len: u32`, `content: Vec<DiffOp>`
- [ ] `Delta` — `id: DeltaId`, `file: FileNode`, `diff: LineDiff`, `source: SourceType`, `timestamp: i64`
- [ ] `Snapshot` — `id: SnapshotId`, `file: FileNode`, `deltas: Vec<DeltaId>`, `parents: Vec<SnapshotId>`, `partition_type: PartitionType`, `created_at: i64`
- [ ] `Snapshot::compute_id()` — 根据内容计算 Blake3 哈希

### 1.4 可变实体定义 [1h]

按 [核心数据模型 §2.3-2.5](../architecture/02-核心数据模型.md) 定义：

- [ ] `PartitionType` 枚举（参考文档中的完整枚举）
- [ ] `Partition` — `id: PartitionId`, `name: String`, `current_snapshot: SnapshotId`, `history: Vec<SnapshotId>`, `partition_type: PartitionType`
- [ ] `Layer` — `layer_type: LayerType`, `partitions: Vec<PartitionId>`
- [ ] `LayerType` 枚举

### 1.5 SQLite 建表与迁移（`storage/migrations.rs`）[1h]

按 [Rust实现方案 §7.6](../architecture/07-Rust实现方案.md) 建表：

- [ ] 建表：`file_nodes`, `deltas`, `snapshots`, `partitions`, `partition_history`, `layers`
- [ ] 数据库初始化 + 版本迁移框架
- [ ] `INSERT ONLY` 约束检查（不可变实体拒绝 UPDATE/DELETE）

### 1.6 存储 trait 定义（`storage/repository.rs`）[1h]

按 [Rust实现方案 §7.3](../architecture/07-Rust实现方案.md) 定义异步 trait：

- [ ] `SnapshotStore` trait — `store_snapshot`, `get_snapshot`, `find_snapshots`
- [ ] `DeltaStore` trait — `store_delta`, `get_delta`
- [ ] `PartitionStore` trait — `create_partition`, `update_pointer`, `get_partition`

### 1.7 SQLite 存储实现（`storage/sqlite_storage.rs`）[3h]

- [ ] 实现 `SnapshotStore` for `SqliteStorage`
- [ ] 实现 `DeltaStore` for `SqliteStorage`
- [ ] 实现 `PartitionStore` for `SqliteStorage`
- [ ] 事务支持（所有写操作在事务内）
- [ ] BLOB 读写（rkyv/serde_json 序列化）
- [ ] 批量查询优化

### 1.8 全局错误类型（`error.rs`）[0.5h]

- [ ] `StorageError` — `NotFound`, `ConstraintViolation`, `Serialization`, `Database`
- [ ] 从 `rusqlite::Error`、`rkyv` 错误的转换
- [ ] `Result<T>` 别名
