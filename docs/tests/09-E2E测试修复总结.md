### 2. 换行符处理问题 ✓
**问题**: `apply_deltas` 函数没有保留原始内容的尾随换行符

**根本原因**:
- `apply_deltas` 函数使用 `lines().join("\n")` 来重建内容
- Rust 的 `lines()` 方法在处理尾随换行符时不会创建额外的空行
- 导致尾随换行符丢失

**修复方案**:
- 修改 `apply_deltas` 函数，检测原始内容是否以换行符结尾
- 在重建时添加换行符（如果原始内容有）

**代码修改**:
```rust
pub fn apply_deltas(content: &str, deltas: &[Delta]) -> Result<String> {
    let has_trailing_newline = content.ends_with('\n');
    let mut lines: Vec<String> = content.lines().map(|l| l.to_string()).collect();

    for delta in deltas {
        lines = apply_line_diff(&lines, &delta.diff)?;
    }

    let result = lines.join("\n");
    if has_trailing_newline {
        Ok(result + "\n")
    } else {
        Ok(result)
    }
}
```
